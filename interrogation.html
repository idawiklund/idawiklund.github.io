Bar application


An Android application created by me and four other classmates that uses Google's API to get information about different bars and clubbs. The application is created with Javascript, React Native.

The application uses three API:s of Google (Google Places, Google Geolocation, Google Maps) to provide the user with information about different bars and clubs from specific locations. The user can choose to give his current location with the application or enter a location. Then different bars and clubs on that location appears. It displays inforamtion about each bar or club, including address, opening hours, pictures from the place, ratings and phone number.
